Bước 1 — Tải các file PDF

Hỗ trợ PDF và Word (.docx). Kéo thả, chọn nhiều file hoặc cả thư mục.

Kéo thả PDF / Word hoặc cả thư mục vào đây hoặc dùng nút bên dưới

Bước 2 — Chọn trường cần thay đổi

Bật công tắc ở trường muốn đổi, nhập giá trị mới. Giá trị này áp dụng cho tất cả file (bước sau vẫn sửa riêng từng file được).

AI đọc câu lệnh + nội dung từng file rồi điền giá trị mới vào đúng file (hiểu cả điều kiện như “file của Sacombank”). Xem lại ở bước 3 trước khi xuất.

Quy tắc thay thế tự do (tuỳ chọn)

Dùng cho phần khác (số hợp đồng, tháng đối soát, tên đơn vị…). Tool tìm đúng chữ bên trái, thay bằng chữ bên phải, trên mọi file. Lưu ý: chữ cần tìm phải nằm gọn trong một dòng.

💡 Mẹo: Nếu chỉ cần đổi số tài khoản / ngân hàng cho tất cả file — chỉ bật đúng trường đó và nhập giá trị mới là xong.

Bước 3 — Xem trước & sửa riêng (nếu cần)

Mỗi dòng là một file. Ô vàng là giá trị sẽ thay đổi. Sửa trực tiếp từng ô nếu một file nào đó cần khác.

Ô vàng = giá trị mớiÔ trắng = giữ nguyên

Xuất file

Tool sửa và trả file đúng định dạng gốc (PDF→PDF, Word→Word), giữ nguyên bố cục.

PDF và Word đều được sửa giữ nguyên định dạng gốc. File PDF có thể chuyển thêm sang Word nếu cần. Font tiếng Việt nhúng sẵn — không cần Internet.

Tải Biên bản đối soát (PDF)

Tool đọc bảng giao dịch, thêm cột Đơn giá trước cột Tổng phí và tự canh cho gọn 1 trang.

Kéo thả Biên bản đối soát (PDF) hoặc cả thư mụchoặc dùng nút bên dưới

Đơn giá & xuất file

Nhập đơn giá áp dụng cho tất cả file; có thể sửa riêng từng file ở bảng dưới.

Xuất file

Bảng được dựng lại để thêm cột nên chữ ký số (nếu có) sẽ mất hiệu lực — cần ký lại sau khi xuất.

Quét mẫu mới

Tải lên 1 file mẫu (PDF/Word). Tool quét cấu trúc và ghi nhớ để dùng lại. Lúc xử lý, bạn chọn mẫu là tool chỉnh theo đúng mẫu đó (ĐNTT nhớ bộ trường; BBĐS nhớ bản đồ cột & vị trí chèn Đơn giá). Mẫu lưu ngay trong trình duyệt, không cần mạng.
